This page demonstrates an XMLUI app embedded as a style-isolated "island" inside a host page with its own CSS. The community calendar below lives in a Shadow DOM — its styles don't affect this page, and this page's styles don't affect the calendar.

About this demo

The calendar on the left is a full XMLUI app running inside a Shadow DOM. This sidebar proves the host page's styles are intact — serif font, cream background, brown headings.

How it works

  • Add data-shadow to the root div
  • XMLUI styles move into the Shadow DOM automatically
  • Host page CSS is unaffected

About XMLUI themes

XMLUI has a powerful theming system that you can use to match the embedded app's look and feel to the host page.
