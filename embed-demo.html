Community Calendar — Embed Demo

This page is a test harness for embedding the community calendar inside another site via an iframe that auto-grows to fit its content, so the host page owns all scrolling and pagination ("Later") stays reachable.

Production-path test. The iframe below opts into auto-height via ?autoheight=true. The sender lives inside xmlui/index.html (gated on the query param), and the only parent code needed is the tiny message listener at the bottom of this page.

Some filler text above the iframe so you can tell the host page is doing the scrolling, not the iframe. Scroll down through the calendar: you should be able to reach the Later button at the bottom without getting stuck inside a nested scroll container.

Filler text below the iframe. If auto-resize is working, you'll see this paragraph only after scrolling past the entire event list — confirming there's exactly one scroll container (the host page), not two.

How it works (once productized):

<iframe id="cc"
        src="https://judell.github.io/community-calendar/xmlui/index.html?city=CITY&embed=true&autoheight=true"
        style="display:block;width:100%;height:0;border:0;visibility:hidden;"
        scrolling="no"></iframe>
<script>
addEventListener('message', function (e) {
  if (e.origin !== 'https://judell.github.io') return;
  if (e.data && e.data.type === 'cc-embed-resize') {
    var f = document.getElementById('cc');
    f.style.height = e.data.height + 'px';
    f.style.visibility = 'visible';
  }
});
</script>
